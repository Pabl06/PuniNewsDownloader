# PuniNewsDownloader
An easy tool to extract the images from the YW Puni Puni News site.

<img width="1536" height="1024" alt="tool_logo" src="https://github.com/user-attachments/assets/9469a5ce-890f-4e92-ab65-d6bb078df57e" />

## How does it work?
Just execute the "YWPP_News_IMG_Downloader.py" script and it will download all the images at the News site in that moment.
To execute it, just open a CMD on the folder or go to the folder route and put this into the CMD:

```bash
python YWPP_News_IMG_Downloader.py
````

## Where I can find the images?
You can find them in the "Assets" folder that the script creates.

## Requirements
- At least Python 3: https://www.python.org/downloads/
